# BTC Quantitative Backtest Strategy

## 策略簡介
結合 MA20/MA60 均線交叉與布林通道的 BTC 量化回測策略，
回測期間涵蓋 2020-2026 年，完整包含一個牛市與熊市週期。

## 策略邏輯
**買入條件（同時符合）：**
- MA20 > MA60（短期均線在長期均線上方，確認上漲趨勢）
- 價格 > 布林通道下軌（未跌破支撐）

**賣出條件（符合其一）：**
- MA20 < MA60（趨勢反轉）
- 價格 > 布林通道上軌（可能過熱）

**參考指標（不影響決策）：**
- MACD：確認動能方向
- RSI：觀察超買超賣

## 回測結果
| 指標 | Buy & Hold | 本策略 |
|------|-----------|--------|
| 報酬率 | 938% | 870% |
| 最大回撤 | -76.6% | -46.6% |
| 夏普值 | 0.91 | 1.11 |

## 結論
策略在熊市期間有效控制回撤，最大回撤從 76.6% 壓縮至 46.6%，
夏普值 1.11 優於市場的 0.91，顯示風險調整後報酬更佳。
ETH 回測結果顯示此策略更適合波動相對穩定的資產。
